# katsana-sdk-ios
KatsanaSDK is a framework for iOS/macOS to access data from Katsana platform.

## Installation

###[Cocoapods](https://cocoapods.org/)

Add these to your Podfile

For iOS
```
use_frameworks!
target "PROJECT_NAME" do
    platform :ios, '8.0'
    pod 'katsana-sdk-ios', '~>0.9.0'
end
```

For macOS
```
use_frameworks!
target "PROJECT_NAME" do
    platform :osx, 10.11'
    pod 'katsana-sdk-ios', '~>0.9.0'
end
```
